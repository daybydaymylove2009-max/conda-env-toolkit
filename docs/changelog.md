# 更新日志

> 📝 Conda 环境管理工具箱的版本历史

## [v0.3.0] - 2026-05-01

### ✨ 新增功能

- **多镜像源支持** - 支持官方主站 + 6 大国内镜像源：
  - 官方主站 (PyPI/Anaconda)
  - 清华大学 TUNA
  - 阿里云
  - 中国科学技术大学 (USTC)
  - 腾讯云
  - 华为云
  - 豆瓣
- **镜像源动态切换** - 恢复时可选择使用主站或镜像源，支持命令行 `--mirror` 参数和交互式菜单
- **实时失败原因分析** - 安装失败时立即显示具体原因：
  - 系统库/开发包识别（libasprintf, libblas, libgcc 等）
  - 网络问题（超时、连接失败）
  - SSL证书验证失败
  - 依赖冲突
  - 权限不足
  - 磁盘空间不足
  - 内存不足
  - Python版本不兼容
  - 包在渠道中未找到
- **并行安装进度显示** - 批量安装时显示实时百分比进度

### 🔧 改进

- 优化交互式菜单镜像源选择界面，使用序号选择
- 改进失败包日志，包含具体失败原因
- 增强包名特征识别算法

## [v0.2.0] - 2026-05-01

### ✨ 新增功能

- **智能跳过** - 恢复时自动跳过已安装且版本相同或更新的包，避免重复安装和版本降级
- **实时进度** - 恢复过程中显示百分比进度条（Conda包每10个更新，Pip包每5个更新）
- **恢复报告** - 恢复完成后生成详细对比报告，包含：
  - 备份包数 vs 当前包数
  - 安装成功/失败/跳过统计
  - 缺失包列表
  - 版本差异列表
  - 额外包列表
  - 恢复完整度百分比
  - 质量评价（优秀/良好/一般/较差）
- **失败分析** - 15+ 类失败原因智能分析：
  - 深度学习框架（torch/tensorflow）
  - Git依赖
  - 编译工具（cython/cmake）
  - 平台特定包
  - 科学计算包（numpy/scipy）
  - 数据库驱动
  - Web框架
  - GUI图形库
  - 图像处理
  - 安全加密
  - 版本冲突
  - 权限问题
  - 已弃用/重命名包
  - 私有/内部包
  - 通用错误（网络、SSL、内存、磁盘、Python版本）
- **日志导出** - 失败包自动保存详细日志到 `_restore_logs/` 目录，包含：
  - 失败原因分析
  - 3-4条针对性解决方案
  - 手动安装命令
  - 通用解决方案
  - 批量安装脚本
- **多备份管理** - 支持同一环境多份备份：
  - 自动按时间排序
  - 默认选择最新版本
  - 提示存在多份备份
- **递归查找** - 自动查找子目录中的备份文件

### 🔧 改进

- 优化备份文件查找逻辑，支持嵌套目录
- 改进错误处理和用户提示
- 增强版本比较算法

## [v0.1.0] - 初始版本

### ✅ 核心功能

- **环境备份** - 支持 JSON/TXT/YAML/Requirements/Markdown 多种格式
- **环境恢复** - 智能恢复，支持并行安装和 UV 加速
- **环境克隆** - 一键克隆环境
- **环境对比** - 分析环境差异
- **环境查看** - 查看环境详细信息
- **系统清理** - 清理 Conda/Pip/UV 缓存和临时文件
- **环境转换** - Conda ↔ UV 无缝转换
- **包分类** - 2000+ 包分类数据库
- **断点续传** - 中断后可继续恢复
- **智能重试** - 自动重试失败安装
- **版本回退** - 安装失败时尝试其他版本

### 🎨 交互体验

- 交互式菜单 + 命令行双模式
- Rich 进度条可视化
- 彩色日志输出
- 配置文件持久化
